# MyWordle 🚀

MyWordle is a fun and interactive Wordle-style browser game built using **HTML, CSS, and JavaScript**.

Players must guess the hidden **5-letter word** within **6 attempts** using logic and clues.

---

## 🎮 Features

* 5-letter word guessing game
* 6 attempts to win
* Color hint system

  * 🟩 Correct letter & position
  * 🟨 Correct letter wrong position
  * ⬜ Wrong letter
* Responsive design for mobile & desktop
* Clean modern UI
* Restart game button
* Smooth gameplay experience

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 🚀 How to Run

1. Download or clone this repository
2. Open project folder in VS Code
3. Run `index.html` using Live Server
